# My Archlinux install script

![alt text](image.png)

# HW configuration:
UEFI Only, BIOS/MBR/Legacy not supported

Program install as:
- Bootloader: Systemd-Boot
- GUI Installer: dialog


# Get script:
1) Install Git:
```
pacman -Sy git
```

2) Clone github repo:
```
git clone https://github.com/trinteen/archinstall
```

3) Enter to directory:
```
cd archinstall
```

4) Run
```
sh start.sh
```
